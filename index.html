Project Title: Neobay

Team Number: 2

Team Members: Najia Shineeb, Danielle Nguyen, Deepali Kaushal, Michele Catani, Dylan Bourque

Description: Customer are able to post and purchase items. This will be an auction style website where users can participate in a bidding process to buy and sell items. There will be registration, login, and user profiles. Furthermore, there will be a search feature, product pages, rating/reviews, a bidding process, a shopping cart, and a checkout process.

Tools used: Next JS, BootStrap (html/css), React (JavaScript), Flask & SQLAlchemy(Python), Sqlite
